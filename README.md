**Algorithms**

*Searching*

Search for a given input element within an array and get back it's location/idex within the array.
- Linear Search
- Binary Search

*Sorting* 

Sorts elements in an array in ascending order.
- Selection Sort
- Bubble Sort
- Insertion Sort
- Merge Sort

*Computing*
- GCD: Finds the greatest common divisor among 2 numbers
